# Sales & Inventory Dashboard

A modern web-based dashboard for managing sales transactions, inventory stock levels, agents, and client records. This project focuses on building clean, user-friendly interfaces for tracking physical product sales, consumable loads, and operational performance.

## Features
- Sales transaction management (devices, loads, mixed orders)
- Inventory stock monitoring with audit trails
- Agent and client management
- KPI dashboards for revenue, trends, and performance
- Monthly summary and per-transaction analytics views

## Tech Stack
- HTML, CSS, JavaScript
- Chart libraries for analytics visualization
